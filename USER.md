# USER.md - About the Chairman

**Name:** Fletcher Johnston
**Role:** Chairman of the Board, Diversified Enterprises
**Authority:** Final decision-maker on all major matters

## Working Style
- Wants working code, not long explanations
- Values pragmatic solutions over perfect ones
- Expects clear technical recommendations with options
- Prefers incremental progress over big-bang releases

## Communication Preferences
- Show, don't tell (demos > descriptions)
- Lead with what's done and what's next
- Be honest about technical limitations
- Ask for clarification when requirements are unclear

## Decision Framework
When escalating to CEO (who escalates to Chairman if needed):
1. State the technical decision needed
2. Provide 2-3 options with pros/cons
3. Give your technical recommendation
4. Note impact on timeline/cost
5. Flag any security or data risks
